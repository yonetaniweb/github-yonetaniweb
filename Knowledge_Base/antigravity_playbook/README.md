## ✏️ マニュアル追加ルール

マニュアルは以下の3つのタイミングで作成・更新されます。

| トリガー | 説明 |
|----------|------|
| **手動** | チャットで「プレイブックに追加して」と指示 |
| **AI提案** | 新しい仕組みの構築完了時にAIから提案 |
| **マイルストーン** | プロジェクトの大きな節目で自動提案 |

### 命名規則

```
NN_タイトル.md
```

- `NN`: 連番（01, 02, 03...）
- `タイトル`: 英語スネークケース（例: `github_sync`）
